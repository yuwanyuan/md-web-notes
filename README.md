## 中文介绍

**mdQuick** 是一款轻量级、注重隐私的 Markdown 笔记应用。它不存储您的任何数据，所有笔记均保存在您的浏览器本地或您自己的 WebDAV 服务器上。

### ✨ 主要特性

- 🔒 **隐私至上**：数据完全由您掌控。支持连接私有 WebDAV 服务器（如坚果云、Infini-Cloud、Nextcloud 等）。
- 📝 **强大的编辑器**：基于 CodeMirror 6 构建，支持实时预览、GFM 语法、代码高亮、数学公式及脚注。
- 🔄 **无缝同步**：支持自动保存和手动同步，确保您的笔记在不同设备间保持一致。
- 🌓 **暗黑模式**：完美适配系统主题，提供舒适的夜间阅读与编辑体验。
- 📱 **响应式设计**：无论是在桌面端还是移动端，都能提供流畅的操作体验。
- 🌐 **多语言支持**：内置中英文界面，自由切换。
- 🚀 **智能加载**：可选“默认打开最新笔记”功能，快速进入工作状态。

### 🛠️ 技术栈

- **前端**: React 19, Vite, Tailwind CSS 4
- **编辑器**: CodeMirror 6
- **图标**: Lucide React
- **动画**: Motion
- **同步**: WebDAV Client

### 🚀 快速开始

1. **配置存储**：打开设置，填写您的 WebDAV 服务器地址、用户名和密码。
2. **开始写作**：点击“新建笔记”即可开始记录。
3. **自动同步**：应用会自动将您的更改同步到 WebDAV 服务器。

## English Introduction

**mdQuick** is a lightweight, privacy-focused Markdown note-taking application. It does not store any of your data; all notes are kept locally in your browser or on your own WebDAV server.

### ✨ Key Features

- 🔒 **Privacy First**: You have full control over your data. Connect to your private WebDAV server (e.g., Jianguoyun, Infini-Cloud, Nextcloud).
- 📝 **Powerful Editor**: Built on CodeMirror 6, supporting real-time preview, GFM syntax, code highlighting, math formulas, and footnotes.
- 🔄 **Seamless Sync**: Supports auto-save and manual synchronization to keep your notes consistent across devices.
- 🌓 **Dark Mode**: Perfectly adapts to system themes, providing a comfortable night reading and editing experience.
- 📱 **Responsive Design**: Provides a smooth operating experience on both desktop and mobile devices.
- 🌐 **Multi-language Support**: Built-in Chinese and English interfaces, switch freely.
- 🚀 **Smart Loading**: Optional "Open latest note by default" feature to quickly resume your work.

### 🛠️ Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS 4
- **Editor**: CodeMirror 6
- **Icons**: Lucide React
- **Animations**: Motion
- **Sync**: WebDAV Client

### 🚀 Quick Start

1. **Configure Storage**: Open settings and fill in your WebDAV server URL, username, and password.
2. **Start Writing**: Click "New Note" to start recording.
3. **Auto Sync**: The app will automatically sync your changes to the WebDAV server.
